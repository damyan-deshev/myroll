# Myroll Agent Notes

- Always source `.venv` before running Python commands that install packages or depend on project Python dependencies. Do not install Python packages into the OS environment directly.
- When a real-provider, synthetic matrix, or model-quality finding is not a backend contract failure, turn it into GM-operable documentation before treating it as closed. Prefer a short FAQ/field-guide entry with screenshots, the visible UI warning, the manual recovery path, and the reason it is not a hard safety failure. These documented scenarios can later become tooltips, toasts, or contextual help links in the UI.
